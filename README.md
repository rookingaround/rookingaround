
**Lᴀᴍᴇɴᴛ**

I'ᴍ ᴅᴏɴᴇ ᴡɪᴛʜ ᴛʜɪs ʙs.

I'ᴍ ᴅɪsᴀᴘᴘᴇᴀʀɪɴɢ ғʀᴏᴍ PT ғᴏʀ ᴀᴡʜɪʟᴇ, ᴅᴜᴇ ᴛᴏ sᴏᴍᴇ ᴜɴᴅᴇʀʟʏɪɴɢ ᴍᴇɴᴛᴀʟ ʜᴇᴀʟᴛʜ ɪssᴜᴇs ᴀɴᴅ sᴏᴍᴇ ᴘᴇᴏᴘʟᴇ ᴡʜᴏ ᴀʀᴇ ᴏɴ ɪᴛ. Mʏ sɪᴛᴇs ᴡɪʟʟ sᴛɪʟʟ ʙᴇ ᴏᴘᴇɴ ʙᴜᴛ I ᴍᴀʏ ɴᴏᴛ ʀᴇsᴘᴏɴᴅ. Tʜᴇʀᴇ ɪs ᴀʟsᴏ ᴀ ᴄʜᴀɴᴄᴇ ɪ sᴛɪʟʟ ᴍᴀʏ ʙᴇ ᴏɴ ʙᴜᴛ ɪ ᴡᴏɴ'ᴛ ʀᴇᴀʟʟʏ ʟᴇᴀᴠᴇ ᴛʜᴇ ɪsʟᴀɴᴅ ᴍᴀᴘ.

I ʀᴇғᴜsᴇ ᴛᴏ ʙᴇ ᴀ ᴘʀᴏʙʟᴇᴍᴀᴛɪᴄ ᴘᴇʀsᴏɴ ᴏʀ ʙᴇ sᴇᴇɴ ᴀs ᴏɴᴇ sᴏ ɪ'ᴍ ᴊᴜsᴛ ᴅɪsᴀᴘᴘᴇᴀʀɪɴɢ ᴜɴᴛɪʟ I ᴄᴀɴ ʙᴇ ᴀ ʙᴇᴛᴛᴇʀ ᴘᴇʀsᴏɴ. I ʜᴏɴᴇsᴛʟʏ ᴍᴀʏ ɴᴏᴛ ʀᴇᴛᴜʀɴ, ᴏʀ ᴊᴜsᴛ ɴᴏᴛ ʀᴇᴛᴜʀɴ ᴏɴ ᴛʜɪs ᴀᴄᴄᴏᴜɴᴛ ᴀɢᴀɪɴ. 

Wʜʏ? 

Sᴏᴍᴇ ᴀʀᴇ ᴘᴇʀsᴏɴᴀʟ ʀᴇᴀsᴏɴs ʙᴜᴛ ᴏᴠᴇʀᴀʟʟ, ɪ ʜᴀᴛᴇ ʜᴏᴡ ɪ ᴘʀᴇsᴇɴᴛ ᴍʏsᴇʟғ ᴏɴ PT. Tʜᴀᴛ's ᴏɴᴇ ᴏғ ᴛʜᴇ ʀᴇᴀsᴏɴs ʙᴜᴛ sᴀɪᴅ ʙᴇғᴏʀᴇ ᴛʜᴇʏ ᴀʀᴇ ᴘᴇʀsᴏɴᴀʟ.

Pʟᴇᴀsᴇ ʀᴇsᴘᴇᴄᴛ ᴍʏ ᴄʜᴏɪᴄᴇs. Sᴀɪᴅ ʙᴇғᴏʀᴇ ɪғ ʏᴏᴜ ᴡᴀɴᴛ ᴛᴏ ᴄᴏɴᴛᴀᴄᴛ ᴍᴇ ᴍʏ sᴛʀᴀᴡᴘᴀɢᴇ ᴀɴᴅ ᴛᴜᴍʙʟʀ ɪs sᴛɪʟʟ ʟɪɴᴋᴇᴅ ᴛᴏ ᴛʜɪs ᴀᴄᴄᴏᴜɴᴛ.
